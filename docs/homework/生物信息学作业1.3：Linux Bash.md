## 学习笔记

- bash是Linux系统的一种shell，可通过交互或非交互（脚本）形式输入命令

- bash基本命令：`touch`新建脚本（注，直接写touch bash.sh即可，$为命令行自带），`chmod`权限；进入vim编辑，`#!/bin/bash`解释器声明语句，必须写在第一行，`echo " "`输出，`exit 0`表示执行成功，可省略；`bash 文件名.sh 或./文件名`执行

- `echo -e 'a1\n1' > a1.txt`测试文件创建和写入，`-e`表示对`\n`转义为换行。详见https://chat.deepseek.com/share/x5hrey4r7pw3gl63yh

- 写入bash脚本处于vim编辑器中，从insert转回terminal模式需要使用esc，然后再通过`:wq`离开vim terminal

- 赋值操作符（=）周围不能有空格。

- 将文件名赋值为变量，可通过变量直接对文件操作。例如

- ```
  file1=a1.txt
  ## 显示a1.txt第一行。注意引用变量需要使用${},对于自定义变量使用$变量名也可以
  head -n 1 ${file1}
  ```

- 关于`cat`命令：直接对文件执行`cat`命令会输出文件内容，但如果使用`cat...>...`会使内容直接重定向到某个文件，不会输出到终端

- 脚本的输入交互：

- ```
  # 提示用户输入一个值
  echo -n "please input a number:"
  
  # 保存用户输入的值到num中
  read num
  ```

- bash条件判断：注意变量的引用方式。特别注意，变量和表示条件表达式始终的`[]`之间必须**有空格**，否则变量与`]`会成为整体而无法被解析。`fi`与`if-elif`语句对应，表示判断结束

- ```
  if [ "$num" -lt "0" ];then
  # 小于0,则输出“negtive number”
  ```

- 调取文件名。注意`ls`作为命令要用斜引号引起来，变量则接收命令的执行结果（command-substitution）。代码执行结果也可用`$(...)`表示

- ```
  # 将ls的结果保存到变量CUR_DIR中
  CUR_DIR=`ls`
  ```

- 关于循环：`done`与`for`语句对应，表示循环结束。但是bash脚本中无缩进也可解析

- 字符串长度：`${#variable}`

- 算数结果`$(())`计算并返回结果，可直接赋值。`(())`同样可进行算数计算，但会直接对变量进行修改

- 连接字符串：直接写在一起，如`s=$s1$s2`



## 作业脚本

#!/bin/bash

cd /home/test/share

touch filename.txt

touch dirname.txt

cd ~/share/bash_homework

NAME=`ls`

for val in $NAME

do

if [ -f $val ];then

cat $val >> filename.txt

elif [ -d $val ];then

cat $val >> dirname.txt

fi

done

cd ../

cat filename.txt > linux_bash_homework.md

cat dirname.txt >> linux_bash_homework.md

exit 0
