# DNA-Seq and Genomic features

## basic facts

- C-value paradox: genome size not correlated with complexity
- GC content less than 50%. but in bacteria GC content varies greatly from 30% to over 70% (may result from extreme living environment)
- in mammalian, GC forms isochore structures especially near genes
- only ~3% of the genome encode for proteins
- Why Amoeba has 670Gbp genome?

- pseudogenes
